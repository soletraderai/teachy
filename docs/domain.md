**Teachy** - An interactive learning app that transforms YouTube videos into active Q&A sessions. Key features include transcript extraction, knowledge base enrichment, AI-generated questions, and a "dig deeper" chat mode. Designed for kinesthetic learners who retain through doing.

## Critical Finding: QuizTube Issues

**QuizTube is NOT recommended** for these reasons:

1. **quiztube.com** - Registered since 2006, expires 2028 (GoDaddy)
2. **Multiple existing products** already using "QuizTube":
   - [QuizTube Chrome Extension](https://chromewebstore.google.com/detail/quiztube-interactive-lear/cahealpimaijmglmbnegeepchojnicbb) - Interactive learning extension

## Competitor Domain Patterns (Research)

From [Smart Branding's EdTech analysis](https://smartbranding.com/global-edtech-unicorns-and-their-domain-choices/):
- **31 EdTech unicorns** - none use hyphens
- Premium domains like **class.com** signal credibility
- **.com** is preferred, **.io** and **.ai** are acceptable alternatives
- Names that describe the action (learn, study, class) perform well

Successful patterns:
- [Edpuzzle](https://edpuzzle.com/) - "Make any video your lesson"
- [Quizizz](https://quizizz.com/) - Free online quiz platform
- [LearnTube.ai](https://learntube.ai/) - Google-backed, 2M+ learners
